# cs.finance
cs.finance is the modern, affordable, easy-to-build-with crypto API platform for crypto builders, traders, and startups. cs.finance aggregates price data across the crypto ecosystem with data feeds available via real-time WebSocket and REST APIs.

### Software Highlights
- More than 40 GitHub repositories, including deployed projects, libraries and more
- Microservices architecture and realtime event streaming via Kafka with Redpanda
- Running on Spring Boot, Angular, and AWS for long-term scalability
